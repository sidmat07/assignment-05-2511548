# Customer Churn Neural Network Analysis 

## Project Objective
Build and analyze a neural network model to predict customer churn using TensorFlow/Keras.

---

# Dataset
- Dataset Name: customer_churn_nn.csv
- Target Variable: churn
- Churn = 1 → Customer Churned
- Churn = 0 → Customer Retained

---

# Tasks Completed

## Task 1: Dataset Understanding
- Dataset exploration
- Missing value check
- Statistical summary
- Target variable distribution

## Task 2: Data Preprocessing
- Removed unnecessary columns
- Encoded categorical variables
- Feature scaling using StandardScaler
- Train-test split

## Task 3: Neural Network Model Building
- Built feed-forward neural network
- Used hidden layers with ReLU activation
- Used sigmoid output layer

## Task 4: Training and Evaluation
- Model training
- Accuracy and loss evaluation
- Confusion matrix
- Classification report

## Task 5: Hyperparameter Experimentation
- Tested multiple model configurations
- Compared accuracy across experiments

## Task 6: Final Reflection
- Weights and biases
- Activation functions
- Learning rate impact
- Underfitting vs overfitting
